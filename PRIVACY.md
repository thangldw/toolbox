## Tiếng Việt

Các ứng dụng trong Thang Toolbox mặc định xử lý dữ liệu cục bộ.

**Diskora** không có telemetry, analytics, quảng cáo hoặc tài khoản người dùng. Danh sách file, ảnh, hash, lịch sử dọn dẹp và kết quả phân tích không được gửi tới máy chủ. Dữ liệu hoạt động được lưu tại `~/Library/Application Support/Diskora`.

**Changeora** không có telemetry, analytics, quảng cáo, tài khoản hoặc network client. Snapshot chỉ chứa metadata phục vụ so sánh như đường dẫn, timestamp, kích thước, Bundle ID, version và thông tin chữ ký; app không đọc nội dung tài liệu cá nhân. Dữ liệu được lưu tại `~/Library/Application Support/Changeora`.

GitHub có thể ghi nhận lượt tải release theo chính sách riêng của GitHub. Việc tải source code hoặc binary từ GitHub nằm ngoài phạm vi hoạt động của ứng dụng.

Nếu một ứng dụng tương lai sử dụng mạng hoặc dịch vụ bên thứ ba, thư mục ứng dụng đó phải có tài liệu privacy riêng trước khi phát hành.

## English

Applications in Thang Toolbox process data locally by default.

**Diskora** has no telemetry, analytics, advertising, or user accounts. File lists, photos, hashes, cleanup history, and analysis results are never sent to a server. Operational data is stored in `~/Library/Application Support/Diskora`.

**Changeora** has no telemetry, analytics, advertising, accounts, or network client. Snapshots contain only comparison metadata such as paths, timestamps, sizes, Bundle IDs, versions, and signature information; the app does not read personal document contents. Data is stored in `~/Library/Application Support/Changeora`.

GitHub may record release downloads under GitHub's own policies. Downloading source code or binaries from GitHub is outside the applications' runtime behavior.

If a future application uses networking or a third-party service, its application directory must include a dedicated privacy document before release.

## 日本語

Thang Toolbox のアプリケーションは、デフォルトでデータをローカル処理します。

**Diskora** には telemetry、analytics、広告、ユーザーアカウントがありません。ファイル一覧、写真、ハッシュ、クリーンアップ履歴、分析結果がサーバーへ送信されることはありません。動作データは `~/Library/Application Support/Diskora` に保存されます。

**Changeora** には telemetry、analytics、広告、アカウント、network client がありません。snapshot には、パス、timestamp、サイズ、Bundle ID、version、署名情報など、比較に必要な metadata のみが含まれます。個人文書の内容は読み取りません。データは `~/Library/Application Support/Changeora` に保存されます。

GitHub は独自のポリシーに基づいてリリースのダウンロード数を記録する場合があります。GitHub からのソースコードまたはバイナリのダウンロードは、アプリケーションの実行時動作の範囲外です。

将来のアプリケーションがネットワークや外部サービスを使用する場合は、リリース前にそのアプリケーション用のプライバシー文書を用意する必要があります。
