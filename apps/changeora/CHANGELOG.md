## Tiếng Việt

Mọi thay đổi đáng chú ý của Changeora được ghi tại đây. Dự án sử dụng Semantic Versioning.

### 1.0.0 — 2026-07-21

#### Added

- Snapshot trước và sau quá trình cài hoặc cập nhật ứng dụng.
- Theo dõi ứng dụng, background services, privileged helpers và system extensions.
- Diff engine với ba mức rủi ro và attribution theo bundle/signature metadata.
- Lịch sử local, khôi phục phiên đang theo dõi và export Markdown.
- Giao diện tìm kiếm, lọc, mở kết quả trong Finder và công khai phạm vi quan sát.
- Smoke test, build scripts, release workflow và icon Changeora.

<a id="en"></a>

## English

All notable Changeora changes are documented here. The project follows Semantic Versioning.

### 1.0.0 — 2026-07-21

#### Added

- Before-and-after snapshots for application installation or update sessions.
- Observation of applications, background services, privileged helpers, and system extensions.
- A diff engine with three risk levels and attribution from bundle/signature metadata.
- Local history, active-session recovery, and Markdown export.
- Search, filtering, Finder reveal, and transparent coverage information.
- Smoke tests, build scripts, a release workflow, and the Changeora icon.
