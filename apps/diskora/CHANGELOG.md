# Changelog

Mọi thay đổi đáng chú ý của Diskora được ghi tại đây. Dự án sử dụng Semantic Versioning.

## 1.0.0 — 2026-07-21

### Added

- Phân tích dung lượng, treemap và so sánh giữa các lần quét.
- Dọn nhanh và dọn chuyên sâu theo mức rủi ro.
- Tìm duplicate bằng SHA-256 và cảnh báo tên dễ nhầm.
- Nhóm ảnh tương tự bằng Apple Vision.
- Gỡ ứng dụng cùng dữ liệu còn sót.
- Phân tích runtime developer và file cấu hình dự án.
- Lịch sử dọn dẹp, báo cáo và khôi phục qua Trash.
- Icon, metadata và nhận diện Diskora.
